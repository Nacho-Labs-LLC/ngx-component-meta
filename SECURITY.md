# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ngx-component-meta, please report it responsibly.

**Email:** [hello@nacholabs.net](mailto:hello@nacholabs.net)

Please **do not** open a public GitHub issue for security vulnerabilities.

## What to expect

- Acknowledgment within 48 hours
- A fix or mitigation plan within 7 days for confirmed issues
- Credit in the release notes (unless you prefer to remain anonymous)

## Scope

This policy covers the `@nacho-labs/ngx-component-meta` npm package and related tooling in this repository.
